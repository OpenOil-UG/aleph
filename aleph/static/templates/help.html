OpenOil Search

This page lets you search OpenOil's collection of more than 1 million corporate filings related to the oil, gas and mining industries.

We have indexed the full text of contracts, company disclosures, news articles and government reports, allowing you to simultaneously check documents from disparate sources.

Much of this text was previously locked up inside image-only PDFs and other inaccessible formats. By using Optical Character Recognition, we have enabled searches across these files for the first time.

Getting Started

Type your search term in the box at the top. To start exploring, you could:
  • Look up a project, such as Tanzania's Bulyanhulu gold mine
  • Find auditors' expressions of concern
  • Compare statements of reserves

Search Syntax

Advanced search queries are possible, based on ElasticSearch query string syntax. For example:
  • ~10 = Proximity: put sign at the end of a quote to define the number of words in a string, that need to contain all words of the quote
  • .* = search for any other character following
  • ? = wildcard search, using ? to replace a single character
  • ~ = fuzziness search, i.e. quikc~
  • many more, see full documentation

Tips

  • Documents containing Malawi, but not Zambia: +Malawi -Zambia
